# 全参加者の最終提出ソース

1BRC for traP の各参加者の最終提出ソースコードです。コンテスト終了後に運営が公開した API (`/api/v1/submissions/{id}/source`) から取得しました。**各ファイルの著作権はそれぞれの参加者にあります**。ここには感想戦・解法比較のための参照用として置いています。

- `00-GPT-5.6-Sol-Ultra-xHigh.c` — 運営が裏コンテストとして GPT-5.6 Sol Ultra (xHigh) に改善させた参考解 (順位外)。全行に日本語注釈と実測値が付いた解説版
- `NN-username.ext` — Private leaderboard の順位 `NN` 順
- `99-username.ext` — レギュレーション違反により失格となった提出
- `leaderboard-public.json` / `leaderboard-private.json` — 最終リーダーボードの生データ (`/api/v1/leaderboard`)。順位・時間・順位変動・提出 ID を含む (API 停止後の参照用)

順位・失格理由などの結果は [`../retrospective.md`](../retrospective.md) を参照してください。
